# Soul & Identity

## The Project's Soul

Terminal-first access to the Gentoo Linux installation handbook. No browser, no GUI — just the handbook rendered as it was meant to be consumed: in the terminal where Gentoo lives.

## Guiding Principles

1. **Terminal-native**: The handbook is a terminal document. It belongs in the terminal, not in a browser tab competing with your build session.
2. **Zero-distraction reading**: Content is king. No ads, no navigation chrome, no noise. Just the handbook and you.
3. **Self-contained**: Works offline. No network calls after initial fetch. The handbook is embedded in the binary.
